# Amazon_sales_analysis_using_powerBI
## Project Overview
This project analyzes Amazon sales data using Power BI to identify sales trends, product performance, and category-wise insights through interactive dashboard visualizations.

The dashboard was created to practice data visualization, dashboard creation, and business analysis using Power BI.

---

## Tools Used
- Power BI
- Power Query
- Excel

---

## Features
- Sales performance analysis
- Category-wise sales insights
- Product analysis
- Interactive dashboard
- KPI cards
- Filters and slicers

---

## KPIs Used
- Total Sales
- Total Orders
- Total Products
- Average Sales

---

## Visualizations Used
- Bar Charts
- Pie Charts
- Line Charts
- KPI Cards
- Slicers

---

## Dashboard Insights
- Identified top-performing product categories
- Analyzed sales trends over time
- Compared category-wise sales performance
- Visualized customer purchasing patterns
